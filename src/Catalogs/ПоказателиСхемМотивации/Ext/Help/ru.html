Справочник "Показатели схем мотивации"

Данный справочник содержит показатели для схем мотивации работников. На основании значений показателей рассчитываются начисления и удержания. У показателя есть следующие реквизиты:

Идентификатор

Название показателя без пробелов, без специальных символов, начинающееся с буквы. Применяется для ввода формул расчета.

Вид показателя

Вид показателя определяет сущность показателя. Виды могут быть следующими:

Денежный показатель - это значит, что значение показателя деньги и в дальнейшем нужно указывать валюту, в которой указано значение показателя.

Числовой показатель - это значит, что значение показателя число.

Процентный показатель - это означает, что показатель задан в процентах и для расчетов нужно значение переводить в число. Например, если задано значение 50 для процентного показателя, то необходимо в расчетах использовать число 50/100 т.е. 0,5.

Оценочная шкала. Так же, показатель может быть шкалой оценки. Шкала оценки - это таблица из трех столбцов. В первых двух столбцах задается диапазон (С ...  По ... ), а в третьей колонке задается результат (Размер). Оценка по шкале производится следующим образом. Предположим, шкале оценки передается число Х. В оценочной таблице находится строка, в которой  число Х находится между значениями в колонках "С" и "По".  Из найденной строки возвращается число из колонки "Размер".

Например, зададим таблицу для расчета премии с продаж. Премия будет вычисляться как процент от оклада в зависимости от количества продаж. В первых двух колонках зададим диапазоны количества продаж, а в третьей колонке - размер премии в проценте от оклада.

Как работает таблица. Если сотрудник сделал 12 продаж, тогда в соответствии с таблицей ему причитается 100% от оклада. Если сделал 6 продаж, тогда только 50%. И так далее.

Оценочная шкала бывает двух видов. Числовая шкала и процентная шкала. Если шкала процентная, то это значит, что при расчетах нужно значения из колонки "Результат" переводить в число. В качестве нашего предыдущего примера приведена процентная оценочная таблица.

Назначение показателя

Определяет порядок ввода показателя.

Показатели делятся на:

  • Для всей компании (организации) - показатель вводятся всей компании (организации)
  • Для подразделения - показатель вводятся в целом по подразделению
  • Для сотрудника - показатель одного сотрудника

    Порядок ввода показателя

    Этот реквизит говорит о порядке ввода показателя пользователем. Возможны следующие варианты:

  • Вводится только при кадровых изменениях- это значит, что значение показателя будет задано при приеме на работу и может изменяться только при кадровых перемещениях работника и документами "Ввод сведений о плановых начислениях работников организаций", "Ввод сведений о плановых удержаниях работников организаций" и "Ввод сведений о плановых начислениях и удержаниях работников". Например, это может быть значение месячной тарифной ставки (оклада), часового тарифа, нормы выработки и т.д.

  • Ежемесячно перед расчётом зарплаты - это значит, что значение показателя задается в регистре сведений "Значения показателей схем мотиваций" перед начислением зарплаты и его нельзя редактировать при расчете. Значения таких показателей ввоядтся при помощи обработки "Показатели расчета заработной платы".

  • Периодически - это значит, что значения показателя задаются в регистре сведений "Периодические значения показателей схем мотивации" один раз и будут действовать до следующего ввода данного показателя с более поздней датой.  Значения таких показателей вводится только для организаций и подразделений при помощи обработки "Показатели расчета заработной платы".

    Для предопределенных показателей схем мотивации, таких как Норма времени в днях, Время в днях, Сдельная выработка порядок ввода не задается. Такие предопределенные показатели рассчитываются программой автоматически на стадии расчета документов Начисление зарплаты.

    Показатель эффективности

    Данный признак означает, что по этому показателю можно оценивать сотрудника. Если данный признак установлен, то этот показатель будет попадать в отчёт "Анализ показателей" и становится доступным переключатель "плановый показатель"/"фактический показатель", который также используется в отчёте "Анализ показателей". 

    Фактический показатель

    Данный реквизит используется для сопоставления нормативного значения показателя и фактического значения показателя в отчете «Анализ показателей».

    Например, кадровому показателю «План продаж» можно указать как фактический показатель «Объем продаж».

    В форме элемента реквизит «ФактическийПоказатель» становится доступным, только если порядок ввода показателя = «Вводится только при кадровых изменениях» или если порядок ввода показателя = «Периодически».

  • Это тарифная ставка

    Если значение данного реквизита истина, то показатель используется также как и предопределенные показатели тарифные ставки такие как

    • Тарифная ставка часовая
    • Тарифная ставка дневная
    • Тарифная ставка месячная
