Настройка выгрузки данных в 1С:Бухгалтерию 8

В полях «Период с» – «по» укажите период, за который будет производиться выгрузка документов. В поле «Файл данных» укажите имя внешнего файла, в который будет производиться выгрузка данных. Кнопка «Выполнить» позволяет выгрузить данные в указанный в обработке файл.

Объем выгружаемых данных зависит от установок, указанных в поле «Выгружаемые объекты».

Выгрузка данных по отражению зарплаты в регламентированном учете. Должен быть взведен флажок «Отражение зарплаты в регл учете»; при этом будут выгружаться документы «Отражение зарплаты в регл учете», дата которых попадает в указанный период.

Выгрузка данных учета для целей исчисления подоходного налога. Должен быть взведен флажок «Данные учета по подоходному налогу»; при этом  будут выгружены:

  • документы «Подоходный налог - доходы и налоги», дата которых попадает в указанный период;
  • данные из регистров сведений: «Доходы с предыдущего места работы», «Применение вычетов» и «Стандартные вычеты физических лиц поподоходному налогу»;
  • данные из регистров накоплений: «Подоходный налог сведения о доходах», «Подоходный налог расчеты с бюджетом», «Подоходный налог имущественные вычеты», введенные за указанный период.

Данные из регистров накопления будут перенесены в документ «Подоходный налог- доходы и налоги». Дополнительно, по данным движений регистра «Подоходный налог имущественные вычеты» с видом движения «Приход» будет создаваться документ «Подтверждение права на имущественный вычет».

Выгрузка кассовых документов. Должен быть взведен флажок «Кассовые документы»; при этом будут выгружаться следующие документы: «Зарплата к выплате организации», «Приходный кассовый ордер», «Расходный кассовый ордер», «Платежное поручение исходящее», «Депонирование организаций», «Списание депонентов в доходы организаций».

Выгрузка сводных данных. В форме «Настройки параметров учета» на закладке «Обмен данными» необходимо указать режим выгрузки данных по работникам. Если будет выбран вариант  «Выгружать данные сводно (по работникам организации)», то документы по отражению зарплаты в регламентированном учете и кассовые документы будут выгружаться сводными по работникам. Перейти к форме  «Настройки параметров учета» можно по гиперссылке информационной надписи.

Важно! Выгрузка сводных данных не будет произведена, если за указанный период выгрузки введены документы «Расходный кассовый ордер» с видом операции «Выплата заработной платы работнику». В этом случае рекомендуется либо выгружать подробные данные, либо отменить проведение документов  «Расходный кассовый ордер» с указанным видом операции.
