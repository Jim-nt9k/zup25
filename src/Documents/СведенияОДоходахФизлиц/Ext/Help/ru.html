Документ «Сведения о доходах физлиц»

Документ «Сведения о доходах физлиц» предназначен для:

  • выдачи справок о доходов работникам организации;
  • формирования сведений о доходах для представления их в МНС.

При формировании сведений о доходах для представления их в МНС можно зарегистрировать переданные на взыскание в МНС суммы недоудержанного подоходного налога.

Кнопка "Заполнить" позволяет внести в документ всех физических лиц, получавших доходы в организации в указанном налоговом периоде.

Кнопка «Печать» позволяет сформировать текстовый файл для передачи в МНС.

Кнопка «Печать» позволяет получить формы Приложения 7 к закону о подоходном налоге с физических лици реестра справок (при передаче в МНС) для вывода на принтер.

Проведение документа (кнопка «ОК») означает, что данные переданы в МНС.

При формировании печатных форм автоматически проводятся проверки выводимых данных, и в случае недостаточности либо ошибочности данных выдаются диагностические сообщения.
